# Documentation Directory

This directory contains all documentation, datasheets, and reference materials for the Open743_GPIO STM32 demonstration project.

## Structure
- **Examples/**: Sample code and demonstration projects for various STM32 features.
- **Parts/**: Information about hardware components, modules, and boards used in the project.
  - **Modules/**: Details and documentation for core modules (e.g., CoreH743I).
  - **OpenH743I-C/**: Reference links and documentation for the Waveshare OpenH743I-C development board.
  - **STM32H743IIT6/**: Datasheets and purchase links for the STM32H743IIT6 MCU.
- **Src/**: Documentation for source code organization and structure.
- **Web/**: Online resources and repository links related to the project.

## Purpose
This directory is intended to provide all necessary reference materials to understand, build, and extend the project. It is useful for both new users and experienced developers.

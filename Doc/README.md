# Doc

This directory contains documentation, datasheets, reference manuals, and additional resources for the STM32H743 project. It is organized into subfolders for demos, parts, style references, and web resources.

## Subdirectories

- **Demos/**: Documentation and guides for example/demo projects.
- **Parts/**: Information and references for hardware modules and boards.
  - **Modules/**: Details and documentation for core modules (e.g., CoreH743I).
  - **OpenH743I-C/**: Reference links and documentation for the Waveshare OpenH743I-C development board.
  - **STM32H743IIT6/**: Datasheets and purchase links for the STM32H743IIT6 MCU.
- **Src/**: Source documentation and additional notes.
- **Style/**: Coding style guides, CMake templates, and related resources.
- **Web/**: External links and web-based documentation.

## Key Files

- **EXTI_Example.md**: Example and explanation for EXTI (external interrupt) usage.
- **stm32h7xx_nucleo_conf.h**: Board configuration header.

Refer to each subdirectory for more detailed documentation and resources.

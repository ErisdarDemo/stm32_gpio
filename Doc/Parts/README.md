# Parts Directory

This directory contains information and documentation about the hardware components used in the Open743_GPIO STM32 demonstration project.

## Structure
- **Modules/**: Documentation for core modules and expansion boards.
- **OpenH743I-C/**: Reference links and documentation for the Waveshare OpenH743I-C development board.
- **STM32H743IIT6/**: Datasheets and purchase links for the STM32H743IIT6 microcontroller.

## Purpose
The Parts directory helps users identify, source, and understand the hardware used in the project, supporting both assembly and expansion.

---

For detailed module information, see the README.md files in subdirectories.

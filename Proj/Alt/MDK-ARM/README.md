# MDK-ARM Directory

This directory contains project files and startup code for the Keil MDK-ARM toolchain.

## Structure
- **Project.uvoptx / Project.uvprojx**: Keil project files for workspace and build configuration.
- **startup_stm32h743xx.s**: ARM Cortex-M7 startup assembly code for STM32H743 series.

## Purpose
The MDK-ARM directory enables development, debugging, and deployment of the Open743_GPIO project using the Keil MDK-ARM IDE. It provides all necessary files for compiling and running the project on STM32 hardware.

# Utilities Directory

This directory contains utility modules and resources for the Open743_GPIO STM32 demonstration project.

## Structure
- **CPU/**: CPU utilities for performance measurement and system monitoring.
- **Fonts/**: Font resources for display and graphical applications.

## Purpose
The Utilities directory provides reusable modules and resources that enhance the functionality and flexibility of the project. These utilities support advanced features such as performance profiling and custom display output.

---

For main application code, see the Src directory. For drivers, see the Drivers directory.

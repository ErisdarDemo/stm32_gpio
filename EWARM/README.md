# EWARM Directory

This directory contains project files and startup code for the IAR Embedded Workbench (EWARM) toolchain.

## Structure
- **Project.ewd / Project.ewp / Project.eww**: IAR project files for workspace and build configuration.
- **startup_stm32h743xx.s**: ARM Cortex-M7 startup assembly code for STM32H743 series.
- **stm32h743xx_flash.icf**: IAR linker configuration file for STM32H743 flash memory.

## Purpose
The EWARM directory enables development, debugging, and deployment of the Open743_GPIO project using the IAR Embedded Workbench IDE. It provides all necessary files for compiling and running the project on STM32 hardware.

# CMSIS Directory

This directory contains ARM CMSIS (Cortex Microcontroller Software Interface Standard) headers and device files for STM32.

## Structure
- **Device/**: Device-specific files for STM32.
- **Include/**: CMSIS core and compiler headers.

## Purpose
- Provides low-level hardware abstraction and standard interfaces for ARM Cortex-M microcontrollers.

---

For board support and HAL drivers, see sibling directories.
